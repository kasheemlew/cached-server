# Cached Server

## Install

### get this repo

```sh
$ git clone https://github.com/kasheemlew/cached-server
```

### install requirements

```sh
$ pip3 install -r requirements
```

## Usage

```sh
usage: server.py [-h] [-H HOST] [-p PORT]

Simple Cached Server

optional arguments:
  -h, --help            show this help message andexit
  -H HOST, --host HOST
  -p PORT, --port PORT
```
